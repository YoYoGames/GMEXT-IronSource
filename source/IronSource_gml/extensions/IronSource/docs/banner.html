ironsource_banner_init

This function initialises the ironSource SDK for the Banner ad unit.

This function operates asynchronously, which means that it does not immediately return the requested result. Instead, upon completion of the task, it will trigger the Social Async Event.


Syntax:

ironsource_banner_init()




Returns:

N/A


Triggers:

Social Async Event

Key Type Description
type String The value "ironsource_banner_init"

ironsource_banner_create

Creates and displays a banner ad with the specified placement, size and position.

This function operates asynchronously, which means that it does not immediately return the requested result. Instead, upon completion of the task, it will trigger the Social Async Event.


Syntax:

ironsource_banner_create(placement_name, size, bottom)
Argument Type Description
placement_name String The name of the ad placement.
size BannerSize The size of the banner ad.
bottom Boolean The position of the banner ad (false: top, true: bottom).




Returns:

N/A


Triggers:

Social Async Event

If the banner loaded successfully:

Key Type Description
type String The value "ironsource_banner_loaded"


If the banner was clicked:

Key Type Description
type String The value "ironsource_banner_clicked"


If the banner was presented:

Key Type Description
type String The value "ironsource_banner_presented"


If the banner was dismissed:

Key Type Description
type String The value "ironsource_banner_dismissed"


If the application was left due to banner interaction:

Key Type Description
type String The value "ironsource_banner_left_application"


If the banner failed to load:

Key Type Description
type String The value "ironsource_banner_load_failed"
error String The error message.

BannerSize

These constants represent the size of a given banner.

These constants are referenced by the following functions:

  • ironsource_banner_create


Member Description
Ironsource_BannerSize_Banner Represents the size of a normal banner.
Ironsource_BannerSize_Large Represents a banner of a large size.
Ironsource_BannerSize_Rectangle Represents a rectangular banner.
Ironsource_BannerSize_Smart Represents a dynamically sized banner that adapts to the mobile screen.

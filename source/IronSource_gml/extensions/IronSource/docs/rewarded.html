ironsource_rewarded_video_init

This function initialises the ironSource SDK for the Rewarded Video ad unit.

This function operates asynchronously, which means that it does not immediately return the requested result. Instead, upon completion of the task, it will trigger the Social Async Event.


Syntax:

ironsource_rewarded_video_init()




Returns:

N/A


Triggers:

Social Async Event

If the rewarded video is ready to be shown:

Key Type Description
type String The value "ironsource_rewarded_video_ready"


If the banner loaded successfully:

Key Type Description
type String The value "ironsource_rewarded_video_load_failed".
error String The error message.

ironsource_rewarded_video_show

Shows the rewarded video ad for the specified placement if it is available.

This function operates asynchronously, which means that it does not immediately return the requested result. Instead, upon completion of the task, it will trigger the Social Async Event.


Syntax:

ironsource_rewarded_video_show(placement_name)
Argument Type Description
placement_name String The name of the ad placement.




Returns:

Real


Triggers:

Social Async Event

Triggered when the Rewarded Video ad view has opened. Your game will lose focus.

Key Type Description
type String The value "ironsource_rewarded_video_opened".


Triggered when the Rewarded Video ad view is about to be closed. Your game will regain its focus.

Key Type Description
type String The value "ironsource_rewarded_video_closed".


Triggered when the user finished watching the video and should be rewarded.

Key Type Description
type String The value "ironsource_rewarded_video_rewarded".
reward_name String The name of the reward.
reward_amount String The amount the player should be rewarded.


Triggered when the rewarded video ad failed to show.

Key Type Description
type String The value "ironsource_rewarded_video_show_failed".


Triggered when the video ad was clicked. This callback is not supported by all networks, and we recommend using it only if it's supported by all networks you included in your build.

Key Type Description
type String The value "ironsource_rewarded_video_clicked".
